2048

2048 is a single-player sliding tile puzzle video game. The objective of the game is to slide numbered tiles on a grid to combine them to create a tile with the number 2048; however, one can continue to play the game after reaching the goal, creating tiles with larger numbers.

I did this project as a first serious initiation to IA programm. I love to play this game and I wanted to know if I was able to code an IA better than me at the game. The answer is yes ans no, this IA has the potential to be better than me but it will need more power of calcul to process because the Monte Carlo method requires a lot of computation to work. It was still a success because the program reaches the 2048 tile more than 90% of the time.
